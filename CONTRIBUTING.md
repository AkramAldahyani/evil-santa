# Contributing

Thank you for your interest in contributing to Evil Santa.

This project is focused on cybersecurity education, awareness, and defense.

---

## Contribution Rules

Contributions must follow these rules:

- Keep the project safe and educational.
- Do not add credential harvesting.
- Do not add deauthentication attacks.
- Do not add code that harms real networks.
- Do not add instructions for targeting people without consent.
- Keep explanations beginner-friendly.
- Clearly mark all demos as simulated.

---

## Good Contributions

Good contributions include:

- Better documentation
- Clearer diagrams
- Scanner improvements
- Safety warnings
- Setup instructions
- Troubleshooting fixes
- Awareness material
- Translation improvements

---

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test the documentation or code.
5. Submit a pull request.

Use a clear title and explain what you changed.
